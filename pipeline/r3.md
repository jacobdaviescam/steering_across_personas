# R3: Syntactic Invariance

## Motivation

The instruction-variant extraction method uses 5 different phrasings of each pos/neg instruction (e.g., "be very assertive" vs "speak with authority and confidence"). If the resulting vector changes substantially across phrasings, we're capturing surface-level syntactic features rather than the underlying trait. This experiment disentangles semantic content from instruction phrasing.

## What It Measures

- **Within-persona cross-variant similarity**: for a given persona and trait, how similar are the 5 vectors produced by each instruction variant individually? High values mean the trait direction is robust to phrasing.
- **Across-persona same-variant similarity**: for a given trait and variant index, how similar are the vectors across personas? High values mean persona identity dominates over instruction choice.
- **Invariance comparison**: which factor — persona or variant — explains more of the variance in steering vector direction?

## Implementation

- **Inputs**: `outputs/{model}/activations/*.pt` (must have per-sample keys with variant indices, using key format `v{i}_q{j}`)
- **Script**: `pipeline/r3_syntactic_invariance.py`
- **CLI**: `python pipeline/r3_syntactic_invariance.py --activations-dir outputs/gemma-2-27b-it/activations --layer 22`
- **Outputs**: `outputs/{model}/robustness/syntactic/`
  - `syntactic_invariance.json` — per-pair within-persona cross-variant cosine stats
  - `syntactic_by_trait.json` — trait-level aggregate invariance
  - `cross_persona_per_variant.json` — across-persona similarity for each variant
  - `invariance_comparison.json` — within-persona vs across-persona comparison
  - `syntactic_by_trait.png` — invariance scores grouped by trait
  - `invariance_comparison.png` — bar chart comparing within-persona vs across-persona similarity

**Method**: Splits activations by variant index using the `v{i}_q{j}` key format. For each persona×trait, computes 5 separate contrastive vectors (one per variant, using all 20 questions for that variant). Measures pairwise cosine similarity within-persona across variants, and across-persona within the same variant.

## Results

Within-persona across-variant similarity: **0.659 ± 0.139**. Across-persona within-variant similarity: **0.726 ± 0.088**. Deference is the worst trait for syntactic invariance (0.41); warmth, empathy, and honesty are the best (~0.75). Persona identity matters more than instruction phrasing, but there is real syntactic sensitivity — the 5-variant averaging in the main pipeline is doing meaningful denoising.

## Key Questions

1. Are the extracted representations driven by semantic meaning or surface phrasing of the instructions?
2. Which traits are most sensitive to instruction wording, and should those traits use more variants?
3. Does the 5-variant averaging adequately wash out syntactic artifacts?
