# R2: Convergence Analysis

## Motivation

R1 shows that the full set of 100 activation pairs produces stable vectors, but how many pairs are actually needed? If vectors converge at n=20, collecting 100 is wasteful. If they haven't converged by n=100, we need more data. Convergence curves also reveal which traits have inherently higher-dimensional or noisier representations — important for prioritizing data collection in future scaling (R5).

## What It Measures

- **Vector convergence**: cosine similarity between the subset-size vector and the full-data vector, at increasing subset sizes (1, 2, 5, 10, 20, 50, 100, N_max).
- **Transfer-matrix stability**: Adjusted Rand Index (ARI) between the cluster assignment from the subset transfer matrix and the full-data transfer matrix, plus Frobenius distance between the two matrices.
- **Per-trait convergence rates**: which traits converge fastest/slowest, indicating representational complexity.

## Implementation

- **Inputs**: `outputs/{model}/activations/*.pt` (pos/neg activation pairs), `outputs/{model}/vectors/*.pt` (full-data contrastive vectors)
- **Script**: `pipeline/r2_convergence.py`
- **CLI**: `python pipeline/r2_convergence.py --activations-dir outputs/gemma-2-27b-it/activations --vectors-dir outputs/gemma-2-27b-it/vectors --layer 22`
- **Outputs**: `outputs/{model}/robustness/convergence/`
  - `convergence_curves.json` — per-pair cosine to full-data vector at each subset size
  - `convergence_by_trait.json` — trait-level aggregate convergence curves
  - `transfer_stability.json` — ARI and Frobenius distance at each subset size
  - `convergence_curves.png` — convergence curves plotted by trait
  - `transfer_stability.png` — ARI and Frobenius distance vs subset size

**Method**: At each subset size N, randomly samples N activation pairs (without replacement), computes the contrastive vector, and measures cosine similarity to the full-data vector. Also recomputes the full transfer matrix at each N and compares cluster assignments via ARI.

## Key Questions

1. How many activation pairs are needed for stable vectors? (Directly answers: "what if you start with 1 prompt and slowly increase?")
2. Which traits converge slowest, and does that correlate with context-dependence found in R4?
3. At what N does the transfer-matrix cluster structure stabilize?
